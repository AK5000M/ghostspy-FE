# Getting Started

    1. Running
    	```
    	yarn dev
    	```
    2.Build
    	```
    	yarn build
    	```
    3.Server Starting
    	```
    	yarn start
    	```
    4. Lint checking
       ```
       yarn lint
       ```

Open [http://localhost:4200](http://localhost:4200) with your browser to see the result.
